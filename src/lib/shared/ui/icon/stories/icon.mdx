## Формирование имени иконки
Имена иконок формируются в зависимости от того, как они были названы в фигме
если в фигме название "Document_File / comparison", то в библиотеке оно будет "DocumentFileComparison"

## Пример использования

```tsx
import type { FC } from 'react';
import { Icon } from '@g3/ui';

export const MyComponent: FC = () => (
  <div>
    <Icon name="ArrowArrowUp" />
    <Icon name="AlertDone" size={32} />
    <Icon name="NoAccess" size={40} />
  </div>
);
```
